# gulp
Use for health.

## Project setup
```
npm install
```
### Starting build
```
gulp
```
### Cleaning images
```
gulp cleanimages
```
### Build fonts
```
gulp fonts fontsStyle
```
### Cleaning dist
```
gulp cleandist
```
